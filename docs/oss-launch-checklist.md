# OSS Launch Checklist

- [ ] `LICENSE` is Apache-2.0.
- [ ] `CONTRIBUTING.md` includes DCO sign-off instructions.
- [ ] `SECURITY.md` defines private vulnerability reporting.
- [ ] `craft doctor` passes.
- [ ] `craft validate` passes.
- [ ] Catalog generation passes.
- [ ] Markdown link check passes.
- [ ] GitHub Actions pass on a clean checkout.
- [ ] `git ls-files` contains no private workspace files.
- [ ] No `workspace/`, `work-social/`, logs, local settings, memory, sessions, or checkpoints are tracked.
