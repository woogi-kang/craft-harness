# Craft Harness Adapter: Gemini

Use this repository as a multi-runtime harness:

- reusable instructions live in `skills/`
- role-specific agents live in `agents/`
- slash-command source lives in `commands/`
- orchestration examples live in `templates/` and `examples/`

Before large edits, inspect the relevant skill or command. After edits, run:

```bash
./craft doctor
./craft validate
```
