## v0.1 Alpha

- Publish Core and Dev packs.
- Stabilize `craft doctor`, `craft validate`, and `craft catalog`.
- Document runtime adapters for Claude, Codex, Gemini, OpenCode, and OpenHands.
- Add output style presets.

## v0.2 Orchestration Contracts

- Formalize `plan.json` v2 JSON Schema.
- Add contract generation before worker execution.
- Add automatic QA worker injection for success criteria.
- Emit `qa-report.md` and `qa-report.json`.

## v0.3 Registry

- Migrate skill metadata to frontmatter v2.
- Add compatibility and required tool metadata.
- Generate `.agents/skills` compatible registry exports.
- Split optional packs from core.

## v1.0

- Add run ledger and replay/inspect commands.
- Add MCP server surface for catalog and validation.
- Add stable pack versioning and compatibility matrix.
- Publish public docs site.
